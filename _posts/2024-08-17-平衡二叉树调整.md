### LL型

```mermaid
graph TB
root-->a-->b-->c-->Z
a-->X
b-->Y
c-->W
```

变换步骤：`a` 相对 `b` 做 `右旋` 变换

1. `a->left = b->right`
2. `b->right = a`
3. `root->next = b`

```mermaid 
graph TB
root-->b-->c-->Z
c-->W
b-->a-->Y
a-->X
```

### RR型

```mermaid
graph TB
root-->a-->X
a-->b-->Y
b-->c-->Z
c-->W
```

变换步骤：`a` 相对 `b` 做 `左旋` 变换
1. `a->right = b->left`
2. `b->left = a`
3. `root->next = b`

```mermaid
graph TB
root-->b-->a-->X
a-->Y
b-->c-->Z
c-->W
```

### LR型

```mermaid
graph TB
root-->a-->b-->Y
a-->X
b-->c-->Z
c-->W
```

分步变换：
1. `b` 相对 `c` 做 `左旋` 变换
2. `a` 相对 `c` 做 `右旋` 变换


```mermaid
graph TB
root-->a-->c-->b-->Y
a-->X
b-->Z
c-->W
```

```mermaid
graph TB
root-->c-->b-->Y
b-->Z
c-->a-->W
a-->X
```

一步到位法:
1. `b->right = c->left`
2. `c->left = b`
3. `a->left = c->right`
4. `c->right = a`
5. `root->next = c`

### RL型

```mermaid
graph TB
root-->a-->X
a-->b-->c-->Z
b-->Y
c-->W
```

分步变换：
1. `b` 相对 `c` 做 `右旋` 变换
2. `a` 相对 `c` 做 `左旋` 变换


```mermaid
graph TB
root-->a-->X
a-->c-->Z
c-->b-->W
b-->Y
```


```mermaid
graph TB
root-->c-->a-->X
a-->Z
c-->b-->W
b-->Y
```

一步到位法：
1. `a->right = c->left`
2. `c->left = a`
3. `b->left = c->right`
4. `c->right = b`
5. `root->next = c`
